Android Application Components

Activities

Activities represent the different screens in an Android application. learn more

Fragments

Fragments are reusable UI components that can be combined to create flexible layouts. learn more

Services

Services are background tasks that run independently of the UI. learn more

Intents

Intents are messages that allow components to request functionality from other components, such as starting an activity, broadcasting a message, or invoking a service. Intents can be explicit (targeting a specific component) or implicit (allowing the system to determine the appropriate component). learn more

Permissions

Android apps require permissions to access sensitive device resources like the camera, location, contacts, etc. Developers must declare these permissions in the manifest file and request them at runtime on devices running Android 6.0 (API level 23) or higher. learn more

Adapters and RecyclerViews

Adapters are used to populate views, such as lists or grids, with data from a data source. RecyclerView is a more flexible and efficient replacement for ListView and GridView, commonly used to display large datasets efficiently. learn more

Data Management

SharedPreferences

SharedPreferences is a lightweight mechanism for storing primitive data types (such as integers, booleans, strings) in key-value pairs. It is commonly used for storing simple app preferences and settings. learn more

File Storage

Android provides APIs for reading from and writing to files in internal and external storage. Internal storage is private to the app and can be used to store sensitive data, while external storage is accessible to other apps and users. learn more

SQLite Database

SQLite is a lightweight relational database management system embedded in Android. It allows developers to create and manage structured data in the form of tables, rows, and columns. SQLite is commonly used for storing structured data locally within an app. learn more

Room Persistence Library

Room is a SQLite object mapping library provided by the Android Jetpack architecture components. It provides an abstraction layer over SQLite database operations, simplifying database access and management through the use of annotated Java or Kotlin data objects. learn more

Content Providers

Content Providers are Android components that allow apps to share data with other apps securely. They provide a standard interface for accessing and managing data across different applications and can be used to expose data from SQLite databases or other data sources. learn more

Network Communication

Android apps often need to communicate with remote servers to fetch or send data over the network. This is typically done using HTTP-based networking libraries like Retrofit, Volley, or OkHttp, which handle network requests and responses asynchronously. learn more
