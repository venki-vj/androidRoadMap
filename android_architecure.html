Application Layer
System Apps

These are the applications that come pre-installed on the device by the original equipment manufacturer (OEM) and vendor.

Third-party Apps

These are the applications installed by the user from app stores such as Google Play Store or third-party sources.

Framework Layer
Activity Manager

Manages the lifecycle of applications and provides a common navigation backstack.

Notification Manager

Manages all notifications in the system, including those triggered by the applications.

Package Manager

Manages the application packages installed on the device.

Window Manager

Manages the windowing infrastructure and provides a high-level abstraction for drawing views on the screen.

Content Provider

Manages access to a structured set of data, typically stored in a SQLite database.

View System

Provides the basic building blocks for user interface components.

Android Runtime Layer
Dalvik VM

The Dalvik Virtual Machine is a specialized virtual machine designed specifically for Android to run applications written in Java. It uses a just-in-time (JIT) compilation technique to optimize performance.

Zygote

Zygote is a process in Android that serves as the parent for all app processes. It's responsible for initializing the Dalvik VM and preloading essential libraries and resources to accelerate app startup.

Core Libraries

The Core Libraries include essential Java libraries adapted and optimized for Android. These libraries provide fundamental functionality for Android app development, such as data structures, utilities, and networking.

Libraries Layer
Media

The Media library provides APIs for recording and playing audio and video, including support for various formats and streaming.

Graphics

The Graphics library offers tools for rendering 2D and 3D graphics, including support for hardware acceleration and OpenGL ES.

SQLite

The SQLite library provides a lightweight relational database management system embedded in the Android OS, useful for storing structured data locally.

OpenGL

OpenGL (Open Graphics Library) is a cross-language, cross-platform API for rendering 2D and 3D vector graphics. It's often used for high-performance rendering in games and other graphics-intensive applications.

SSL

The SSL (Secure Sockets Layer) library provides cryptographic protocols for secure communication over a computer network. It ensures data integrity, authentication, and confidentiality.

FreeType

FreeType is a software development library that provides tools for rendering fonts. It's commonly used for text rendering in graphical user interfaces and applications.

Kernel Layer
Wi-Fi Driver

The Wi-Fi driver enables the device to connect to wireless networks and communicate over Wi-Fi.

Display Driver

The display driver controls the display hardware, facilitating the rendering of graphics and text on the screen.

Audio Driver

The audio driver manages audio input and output, including playback of multimedia content and communication through voice calls.

USB Driver

The USB driver enables communication between the device and external USB peripherals, such as keyboards, mice, and storage devices.

Camera Driver

The camera driver controls the camera hardware, allowing the device to capture photos and videos.

Bluetooth Driver

The Bluetooth driver enables wireless communication between the device and other Bluetooth-enabled devices, such as headphones, speakers, and wearables.
